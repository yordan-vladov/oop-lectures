# Условни оператори, цикли

---
## Какво ще научим днес?

- Условни оператори (`if`, `else if`, `else`, `switch`)
- Цикли (`for`, `while`, `do...while`)
- Разширени форми на `for`
- Разлики с Python
- Практически примери и задачи
---
### `if` проверка

```cpp
if (условие) 
{
    // код, ако е вярно
}
```
---
```cpp
int x = 10;
if (x > 0) 
{
    cout << "Положително число";
}
```
---
### `if - else` проверка

```cpp
if (x % 2 == 0) 
{
    cout << "Числото е четно";
} else {
    cout << "Числото е нечетно";
}
```
---
### `if-else if-else` проверка

```cpp
if (grade >= 90) 
{
    cout << "Отличен";
} 
else if (grade >= 75) 
{
    cout << "Много добър";
} 
else {
    cout << "Нужда от повече учене";
}
```
---
### `switch` оператор

```cpp
int day = 3;
switch (day) 
{
    case 1: cout << "Понеделник"; break;
    case 2: cout << "Вторник"; break;
    case 3: cout << "Сряда"; break;
    default: cout << "Невалиден ден";
}
```

### `for` цикъл (основен)

```cpp
for (int i = 0; i < 5; i++) 
{
    cout << i << " ";
}
```
---
### `while` цикъл

```cpp
int i = 0;
while (i < 5) 
{
    cout << i << " ";
    i++;
}
```
---
### `do...while` цикъл

```cpp
int i = 0;
do 
{
    cout << i << " ";
    i++;
} while (i < 5);
```
---
### Управление на цикли

- `break` — прекратява цикъла
- `continue` — прескача текущата итерация
---
```cpp
for (int i=0; i<5; i++) 
{
    if (i==2) continue;   // пропусни 2
    if (i==4) break;      // спри при 4
    cout << i << " ";
}
```
---
### Вложени цикли

```cpp
for (int i=1; i<=10; i++) 
{
    for (int j=1; j<=10; j++) 
    {
        cout << i*j << "\t";
    }
    cout << endl;
}
```
---
### `for` със стъпка различна от 1

```cpp
// Четни числа от 0 до 20
for (int i = 0; i <= 20; i += 2) 
{
    cout << i << " ";
}

// Стъпка k (въведена от потребителя)
int k; cin >> k;
for (int i = 0; i <= 100; i += k) 
{
    // ...
}
```
---
### Обратно броене (отгоре надолу)
```cpp
for (int i = 10; i >= 0; --i) 
{
    cout << i << " ";
}
```
---
### Няколко израза за инициализация/инкремент

```cpp
for (int i = 0, j = 10; i <= j; ++i, --j) 
{
    // i расте, j намалява
    cout << i << "," << j << "\n";
}
```
---
### Пропускане на части от заглавието

```cpp
int i = 0;                 // инициализация извън for
for ( ; ; ) // без условие и без инкремент → безкраен цикъл
{              
    if (i >= 5) break;     // излизаме ръчно
    cout << i++ << " ";
}

// Само условие
for (; i < 10; ) 
{
    // ...
    i += 2;                // инкремент вътре
}
```

---
### Често срещан капан с `size_t` (беззнаков тип)

```cpp
// Безкраен цикъл
for (size_t i = 10; i >= 0; i--) 
{
    cout << i << " ";
}
```

---
### Кома оператор и сложни обновявания

```cpp
for (int a=1, b=1; a <= 8; a+=a, b+=2) 
{
    cout << a << ":" << b << "\n"; // 1:1, 2:3, 4:5, 8:7
}
```

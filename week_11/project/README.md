# Как да компилираш и стартираш

1. Отиди в директорията на проекта:

   ```bash
   cd project
   ```

2. Създай build директория:

   ```bash
   mkdir build && cd build
   ```

3. Генерирай проекта с CMake:

   ```bash
   cmake ..
   ```

4. Компилирай:

   ```bash
   cmake --build .
   ```

5. Стартирай:

   ```bash
   ./MainApp
   ```

### Защо C и C++

- Основи на програмирането: ниско ниво, близо до хардуера
- C – „баща“ на много езици (C++, C#, Java, Python части)
- C++ – мощен език: системно програмиране, игри, софтуер с висока производителност
- Учим се да мислим за **памет, типове и ефективност**
---
### Припомняне - Какво е програма?

- Последователност от инструкции, които компютърът изпълнява
- Написваме **изходен код** → превръща се в **машинен код**
- В Python: интерпретация
- В C/C++: компилация
---
### Интерпретатор vs Компилатор

- **Python**: интерпретатор (ред по ред)
- **C/C++**: компилатор (превежда целия код в изпълним файл)
- Предимства:
    - По-бързо изпълнение (C/C++)
    - По-лесна разработка (Python)
---

| Елемент    | Python               | C++                   |
| ---------- | -------------------- | --------------------- |
| Типове     | Динамични            | Статични              |
| Вход/изход | `print()`, `input()` | `cout`, `cin`         |
| Структура  | Без `main()`         | Задължителна `main()` |
| Компилация | Няма                 | Да (`g++ file.cpp`)   |

---
### C++ - структура на програма

```cpp
#include <iostream>
using namespace std;

int main() 
{
    cout << "Hello, World!" << endl;
    return 0;
}
```

---
- `#include <iostream>` – библиотека за вход/изход
- `main()` – входна точка
- `cout` – извеждане на текст
---
### Изпълняване на програма

1. Създаваме файл `hello.cpp`
2. Пишем кода
3. Компилираме:
```bash
g++ hello.cpp -o hello
```
---
4. Стартираме
```bash
./hello
```

print("Hello, World!")
```
---
**C++**

#include <iostream>

```

### Основни типове данни в C++

- Цели числа: `int`
- Дробни числа: `float`, `double`
- Символ: `char`
- Булеви: `bool` (true/false)
- Низове: `string` (библиотека `<string>`)
---
### Променливи – разлика с Python

- **Python**
```python
x = 5
y = "text"
```
---
- **C++**
```cpp
int x = 5;
string y = "text";
```
---
### Вход/изход в C++

```cpp
#include <iostream>
using namespace std;

int main() 
{
    int a;
    cout << "Въведи число: ";
    cin >> a;
    cout << "Ти въведе: " << a << endl;
    return 0;
}
```
---
### C++ - Чести грешки

- Забравен `;`
- Липсваща фигурна скоба `{ }`
- Не е включена библиотека `<iostream>`
- Грешно име на променлива

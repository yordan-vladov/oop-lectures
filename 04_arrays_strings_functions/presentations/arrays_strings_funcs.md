# Масиви
---
### Какво е масив?

- Масив = последователност от елементи от **един и същ тип**.
- Индексиране започва от **0**.
- Елементите са в **последователна памет**.
---
```cpp
// тип име[размер];
int arr[5];   // масив от 5 цели числа
```
---
### Инициализация на масиви

```cpp
int arr[5] = {1, 2, 3, 4, 5};
int arr2[] = {10, 20, 30}; // размер автоматично = 3
int arr3[5] = {1, 2};      // останалите са 0
```
---
### Достъп и промяна

```cpp
cout << arr[2]; // извежда третия елемент
arr[1] = 3 // променя стойността на втория елемент
```
---
### Обхождане на масив

```cpp
#include <iostream>
using namespace std;

int main() 
{
    int numbers[5] = {2, 4, 6, 8, 10};
    for (int i = 0; i < 5; i++) 
    {
        cout << "Елемент " << i << " = " << numbers[i] << endl;
    }
}
```
---
```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[] = {10, 20, 30, 40, 50};

    for (int value : arr) 
    {
        cout << value << " ";
    }
    cout << endl;
}
```
---
### Двумерни масиви

- Представляват таблица (редове × колони).
```cpp
int matrix[3][4]; // 3 реда, 4 колони
```
---
```cpp
int m[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};
```
---
```cpp
#include <iostream>
using namespace std;

int main() 
{
    int m[2][3] = {{1,2,3}, {4,5,6}};
    for (int i = 0; i < 2; i++) 
    {
        for (int j = 0; j < 3; j++) 
        {
            cout << m[i][j] << " ";
        }
        cout << endl;
    }
}
```

# Низове
---
### Какво е `std::string`?

- Клас за работа с низове от стандартната библиотека.
- По-безопасен и лесен от C-низове.
- Изисква `#include <string>`.
---
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() 
{
    string text = "Hello";
    text += " World!";
    cout << text << endl;
}
```
---
### Основни методи на `std::string`

- `size()` / `length()` → връща дължина
- `empty()` → проверява дали е празен
- `at(pos)` → символ на позиция
- `substr(pos, len)` → подниз
---
- `find("текст")` → връща индекс или `npos`
- `append("текст")` → добавя текст
- `replace(pos, len, "текст")` → замяна
---
### Примери

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() 
{
    string s = "Programming";

    cout << "Дължина: " << s.size() << endl;
    cout << "Буква на позиция 3: " << s.at(3) << endl;

    cout << "Подниз: " << s.substr(0, 6) << endl;
    cout << "Намира 'gram' на позиция: " << s.find("gram") << endl;

    s.replace(0, 6, "Code");
    cout << "След замяна: " << s << endl;
}
```

# Функции
---
### Какво е функция?

- **Функция** = блок код, който изпълнява задача.
- Предимства:
    - Четимост
    - Повторна употреба
    - По-лесна поддръжка
---
```cpp
тип име(параметри) 
{
    // тяло
    return стойност;
}
```
---
- Пример:
```cpp
int square(int x) 
{
    return x * x;
}
```
---
### Прототип, декларация и дефиниция

- **Прототип** (декларация):
```cpp
int sum(int, int);
```
---
- **Дефиниция**:
```cpp
int sum(int a, int b) 
{
    return a + b;
}
```
---
- **Извикване:**
```cpp
cout << sum(3,4);
```
---
### Примери

```cpp
#include <iostream>
using namespace std;

int sum(int, int); // прототип

int main() 
{
    cout << sum(5, 7) << endl;
    return 0;
}

int sum(int a, int b) 
{ // дефиниция
    return a + b;
}
```
---
```cpp
void printArray(int arr[], int size) 
{
    for(int i=0; i<size; i++)
        cout << arr[i] << " ";
    cout << endl;
}

int main() 
{
    int nums[] = {1,2,3,4,5};
    printArray(nums, 5);
}
```

### Основни типове данни в C++

- `int` – цели числа    
- `float`, `double` – реални числа
- `char` – един символ
- `bool` – булева стойност (true/false)
- `string` – низове (библиотека `<string>`)
---
```cpp
int age = 17;
double pi = 3.14;
char grade = 'A';
bool is_student = true;
string name = "Ivan";
```
---
### Допълнителни типове данни

- `short` – по-малък диапазон от `int` (обикновено 2 байта)
- `long` – по-голям диапазон от `int` (обикновено 4 или 8 байта)
- `long long` – още по-голям диапазон (често 8 байта)
- `unsigned` – само положителни стойности, удвоява максималната стойност
---
```cpp
short s = 32000;
long l = 1000000;
long long big = 9000000000;
unsigned int u = 4000000000;
```
---
### Сравнение: `int` срещу `long`

- **int** – стандартен тип за цели числа (най-често 4 байта, диапазон ~ –2 млрд. до +2 млрд.)
- **long** – може да бъде същото като `int` на някои системи, но често е по-голям (64-бита на модерни компилатори)
- **long long** – гарантирано поне 64-бита
---
Важно: точният размер зависи от компилатора и архитектурата (проверява се с `sizeof()`).

```cpp
cout << sizeof(int) << endl;    // напр. 4
cout << sizeof(long) << endl;   // напр. 8
```

### Аритметични оператори

- `+` събиране
- `-` изваждане
- `*` умножение
- `/` деление
- `%` остатък (само при цели числа)
---
```cpp
int a = 7, b = 2;
cout << a / b << endl; // 3
cout << a % b << endl; // 1
```

---
### Логически оператори

- `==` равно
- `!=` различно
- `<`, `>`, `<=`, `>=`
- `&&` логическо И
- `||` логическо ИЛИ
- `!` отрицание
